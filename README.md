# Storage-Optimizer
Storage Optimizer is a desktop utility designed to clean up system clutter and recover disk space. It specifically targets the "duplicate file" problem and "long path" errors commonly found in Windows and OneDrive environments.

# Storage Optimizer Pro
A high-performance Python desktop utility for disk space recovery and file management.

## Technical Features
- **SHA-256 Hashing:** Accurately identifies duplicate files regardless of name.
- **Windows Path Resolution:** Handles extended-length paths (Long Path support).
- **Asynchronous UI:** Multi-threaded Tkinter interface to prevent freezing during scans.

## How to Use
Download the standalone `.exe` from the **Releases** section on the right to run the app without installing Python.
